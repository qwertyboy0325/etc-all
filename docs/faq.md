## 常見問題解決

### 開發環境問題

#### 依賴安裝失敗
```bash
# 清理快取
npm cache clean --force
# 重新安裝
npm install
```

#### 資料庫連接失敗
```bash
# 檢查資料庫服務狀態
docker-compose ps
# 重新啟動資料庫服務
docker-compose restart db
```

#### 前端編譯錯誤
```bash
# 清理建置快取
npm run clean
# 重新建置
npm run build
```

### 部署問題

#### 服務啟動失敗
> 檢查日誌文件，確認配置文件是否正確

#### 性能問題
> 檢查系統資源使用情況，考慮進行性能優化

#### 安全性問題
> 檢查安全配置，確保遵循最佳實踐
